# Listas
## Listas desordenadas
Para generar lista usa   
 \*  
 \-  
 \+


* elemento 1
* elemento 2
+ elemento 3
- elemento 4


## Listas ordenadas
Para generar listas ordenadas usa un numero. por cada item  
1punto item  
1punto item2  
1. item 1
1. item 2
1. item 3

## Sublistas
Para generar una sublista simplementa identa con 4 espacios   en blanco

1. A
1. B
    * B1
    * B2
1. C
